# BHSIS Field App (antigo motoboy-app)

## Objetivo (CRM)
Aplicativo leve para agentes de campo/representantes. Exibe tarefas, visitas e atualizações em tempo real.

## Tecnologias
- React
- Vite
- PWA (service worker simples)
- Axios

## Execução local
```bash
cp .env.example .env
npm install
npm run dev
```

## Porta
- Compose: `4174`.

## Observações
Reaproveita a estrutura do app original; deve ser adaptado para fluxo CRM (visitas, check-ins, atividades).
